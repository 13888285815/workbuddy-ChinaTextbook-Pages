# 中国教材电子课本浏览系统

📚 一个现代化的中国教材PDF在线浏览系统，支持PDF阅读、标注、书签、缩略图等功能。

## 🌐 在线演示

**GitHub Pages**: https://13888285815.github.io/ChinaTextbook-Pages/

## ✨ 功能特性

### 核心功能
- 📖 **PDF在线浏览** - 支持缩放、全屏、适应宽度等多种浏览模式
- 🎨 **标注工具** - 高亮、画笔、文字、橡皮擦等标注功能
- 🔖 **书签管理** - 自动保存阅读进度，支持快速跳转
- 🖼️ **缩略图预览** - 懒加载技术，快速浏览所有页面
- 🔍 **全文搜索** - 按教材名称、学科、版本搜索

### 用户体验
- ⌨️ **快捷键支持** - 方向键翻页、F全屏、B书签等
- 📱 **移动端优化** - 响应式设计，完美支持手机和平板
- 🌙 **护眼主题** - 深色背景，减轻眼睛疲劳
- 💾 **本地存储** - 标注和书签自动保存到本地

## 🚀 快速开始

### 方式一：在线访问
直接访问演示地址即可使用，无需安装。

### 方式二：本地运行
```bash
# 克隆仓库
git clone https://github.com/13888285815/ChinaTextbook-Pages.git
cd ChinaTextbook-Pages

# 使用任意HTTP服务器运行
python3 -m http.server 8080
# 或
npx serve .

# 访问 http://localhost:8080
```

## 🛠️ 技术栈

- **前端框架**: 原生 JavaScript (ES6+)
- **PDF渲染**: PDF.js 6.0.227
- **图标库**: Font Awesome 6.7.2
- **样式**: 原生CSS3（无框架依赖）
- **部署**: GitHub Pages + GitHub Actions

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `←` / `→` | 上一页/下一页 |
| `+` / `-` | 放大/缩小 |
| `0` | 适应宽度 |
| `F` | 全屏模式 |
| `B` | 添加/移除书签 |
| `/` | 搜索教材 |
| `?` | 显示快捷键帮助 |
| `Esc` | 退出全屏 |

## 📝 功能说明

### 标注工具
1. **高亮工具** - 标记重点内容
2. **画笔工具** - 自由绘制
3. **文字工具** - 添加文字注释
4. **橡皮擦** - 擦除标注

所有标注自动保存到浏览器本地存储，下次打开自动恢复。

### 书签功能
- 点击书签按钮保存当前页码
- 下次打开同一教材自动跳转到书签页
- 缩略图中显示书签标识

### Token设置
支持两种Token类型：
- `PRE_高级` - 会员用户
- `BASIC_基础` - 普通用户

（注：当前版本Token仅用于展示用户状态，核心功能免费开放）
